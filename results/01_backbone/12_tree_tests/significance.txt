Significance test of constr vs unconstr trees.

iqtree: http://www.iqtree.org/doc/Command-Reference#tree-topology-tests

For all four combined tree files, the const tree is stored first, then the unconstr one.
Thus, this is also the order in the following result tables.
Results are copied from the end of the IQ-TREE report files *.iqtree

deltaL  : logL difference from the maximal logl in the set.
bp-RELL : bootstrap proportion using RELL method (Kishino et al. 1990).
p-KH    : p-value of one sided Kishino-Hasegawa test (1989).

Plus signs denote the 95% confidence sets.
Minus signs denote significant exclusion.
All tests performed 10000 resamplings using the RELL method.
